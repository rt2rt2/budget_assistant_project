# Бюджетный помощник (интервальные запросы)

Консольная программа для учета ежедневных расходов за один месяц. Пользователь добавляет траты в формате день, сумма, категория и получает сумму расходов за любой период с дня A по день B за O(1)

## Возможности

- Добавление расхода с указанием дня, суммы и категории
- Отмена последнего добавленного расхода
- Сумма расходов за выбранный период с дня A по день B за O(1)
- Поиск дня с максимальным расходом
- Топ-5 категорий по суммарным тратам
- Вывод всех трат по возрастанию суммы

## Используемые структуры данных и алгоритмы

- Префиксные суммы. Массив накопленных расходов обновляется при каждом добавлении и каждой отмене, и сумма за выбранный период считается за O(1)
- Стек для отмены последнего добавленного расхода
- Бинарное дерево поиска для хранения трат. Симметричный обход по BST выводит все траты по возрастанию суммы
- Сортировка вставками для сортировки категорий по сумме трат
- Линейный поиск для дня с максимальным расходом

## Команды меню

После запуска на экран выводится меню, команда выбирается вводом ее номера.

Команда 1 добавляет расход и запрашивает день от 1 до 31, сумму и категорию. 
Команда 2 отменяет последний добавленный расход и убирает его из всех структур. 
Команда 3 считает сумму расходов за период (день A и день B можно вводить в любом порядке). 
Команда 4 показывает день с максимальным расходом. 
Команда 5 выводит топ-5 категорий по тратам. 
Команда 6 выводит все траты по возрастанию суммы. 
Команда 0 завершает работу программы.

## Внутреннее устройство

Класс BST реализует бинарное дерево поиска со вставкой, удалением и симметричным обходом. Класс Node хранит один расход, ключом служит пара из суммы и номера, дополнительно сохраняются день и категория. 
Класс Stack хранит историю добавлений для отмены. 
Массивы daily и prefix отвечают за дневные и накопленные суммы, список expenses содержит все траты, счетчик counter выдает уникальные номера расходам.
